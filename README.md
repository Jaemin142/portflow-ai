# PortFlow AI

## 1. 프로젝트 개요

**PortFlow AI**는 관세청 수출입 데이터를 기반으로 국가·품목별 물류 수요를 예측하고, 공급망 리스크를 조기에 식별하는 데이터 기반 웹서비스입니다.

본 서비스는 국가와 품목군별 수출입 흐름을 시각화하고, 최근 데이터 기반의 3개월 수요예측, 리스크 점수화, What-if 시뮬레이션, TOP 리스크 랭킹 기능을 제공합니다. 이를 통해 수출입 물류 흐름의 이상 징후를 빠르게 파악하고, 공급망 리스크 대응을 위한 의사결정 참고자료로 활용할 수 있도록 설계되었습니다.

항만운영 통계, 공컨테이너 수급 데이터, 선박 입출항 정보는 향후 확장 데이터로 설계했으며, 향후 항만 단위 운영 리스크 분석으로 확장할 수 있습니다.

---

## 2. 문제 정의

글로벌 공급망은 특정 국가, 특정 품목, 특정 물류 경로에 대한 의존도가 높아질수록 외부 충격에 취약해집니다. 수출입 물동량 감소, 특정 국가 수입의존도 증가, 수출입 불균형 확대, 월별 변동성 증가는 물류 수요와 공급망 안정성에 직접적인 영향을 줄 수 있습니다.

그러나 기존의 수출입 통계는 주로 사후 확인용으로 활용되는 경우가 많아, 국가·품목별 리스크를 조기에 파악하고 대응 전략을 검토하는 데 한계가 있습니다.

PortFlow AI는 이러한 문제를 해결하기 위해 다음 질문에 초점을 맞췄습니다.

* 국가·품목별 수출입 흐름이 어떻게 변화하고 있는가?
* 최근 수출입 데이터 기준으로 향후 물류 수요는 어떻게 변할 가능성이 있는가?
* 어떤 국가·품목 조합이 상대적으로 높은 공급망 리스크를 보이는가?
* 수요 변화나 변동성 확대가 발생할 경우 리스크 점수는 어떻게 달라지는가?
* 향후 항만운영 데이터와 결합하면 어떤 방식으로 운영 리스크 분석까지 확장할 수 있는가?

## 3. 핵심 기능

### 3.1 데이터 개요

* 전체 데이터 현황 KPI 제공
* 선택 국가·품목군 기준 데이터 요약
* 수출입 금액, 중량, 무역수지 등 주요 지표 확인
* 선택 조건에 따른 거래 구조 해석 제공
* 원본 데이터 및 월별 집계 데이터 확인 기능 제공

### 3.2 수출입 추이 분석

* 월별 수출입 중량 추이 시각화
* 월별 수출입 금액 추이 시각화
* 월별 무역수지 추이 시각화
* 국가·품목군·분석 지표별 필터링 지원

### 3.3 AI 수요예측

* LinearRegression 기반 3개월 수요예측
* 실제값과 예측값 비교 그래프 제공
* 예측 결과 테이블 제공
* 예측 변화율 기반 해석 문장 제공
* 데이터 기간, 변동성, 예측 변화율을 반영한 예측 주의도 제공

### 3.4 국가·품목 비교

* 선택 품목군 기준 국가별 지표 비교
* 선택 국가 기준 품목군별 지표 비교
* 수출입 중량, 수출입 금액, 무역수지 등 분석 지표별 비교 가능

### 3.5 리스크 분석

* 수요 변화 리스크
* 변동성 리스크
* 수출입 불균형 리스크
* AI 예측 변화 보정
* 국가별 공급망 의존도 분석
* 리스크 원인 Top 요인 자동 해석
* 리스크 등급 기준표 제공
* 대응방안 제시

### 3.6 What-if 시뮬레이션

* 수요 변화율 조정
* 변동성 변화율 조정
* 기준 리스크 점수와 시나리오 적용 후 점수 비교
* 시나리오별 리스크 변화 해석 제공

### 3.7 TOP 리스크 랭킹

* 전체 국가·품목 조합 대상 리스크 계산
* 기존 리스크 점수에 국가별 공급망 의존도 보정 반영
* 최종 리스크 점수 기준 TOP N 랭킹 제공
* TOP 리스크 조합 시각화
* CSV 다운로드 기능 제공

### 3.8 확장 설계

* 현재 구현 범위와 향후 확장 범위 구분
* 향후 확장 데이터 구조 제시
* 항만운영 통계, 공컨테이너 수급 데이터, 선박 입출항 정보 결합 방향 제시
* 수출입 수요 신호에서 항만 운영 리스크 분석으로 확장되는 로드맵 제시

## 6. 향후 확장 설계

PortFlow AI는 현재 관세청 수출입 데이터를 중심으로 구현되어 있으며, 향후 다음 데이터와 결합하여 항만 단위 운영 리스크 분석으로 확장할 수 있습니다.

| 확장 데이터       | 활용 방향                              |
| ------------ | ---------------------------------- |
| 항만운영 통계      | 항만별 물동량, 처리량, 운영 흐름 분석             |
| 공컨테이너 수급 데이터 | 특정 국가·품목 수요 변화에 따른 공컨테이너 부족 가능성 분석 |
| 선박 입출항 정보    | 선박 스케줄, 입출항 흐름, 물류 집중 시점 분석        |

확장 방향은 다음과 같습니다.

1. 수출입 수요 신호 탐지
2. 항만운영 데이터 결합
3. 공컨테이너 및 선박 입출항 데이터 보정
4. 항만 단위 운영 리스크 분석
5. 의사결정 지원 대시보드로 확장

## 7. 리스크 점수 구조

선택 조합의 기본 리스크 점수는 다음 요소를 기반으로 산정됩니다.

```text
선택 조합 리스크 점수
= 수요 변화 리스크
+ 변동성 리스크
+ 수출입 불균형 리스크
+ AI 예측 변화 보정
```

TOP 리스크 랭킹에서는 국가별 공급망 의존도 보정점수를 추가로 반영합니다.

```text
TOP 랭킹 최종 리스크 점수
= 선택 조합 리스크 점수
+ 국가별 공급망 의존도 보정점수
```

리스크 등급은 점수 구간에 따라 낮음, 보통, 높음으로 구분됩니다.

## 10. 실행 방법

### 10.1 패키지 설치

```bash
pip install -r requirements.txt
```

### 10.2 앱 실행

```bash
python -m streamlit run app.py
```

### 10.3 포트 충돌 또는 실행 오류 발생 시

```bash
python -m streamlit run app.py --server.port 8502
```

```bash
python -m streamlit cache clear
```

## 12. 배포 시 주의사항

* `.env` 파일은 GitHub에 업로드하지 않습니다.
* `data/raw/customs_trade.csv` 파일은 앱 실행에 필요하므로 포함합니다.
* README에는 개인 PC 경로를 포함하지 않습니다.
* 현재 버전은 관세청 수출입 데이터를 기반으로 작동합니다.
* 항만운영 통계, 공컨테이너 수급 데이터, 선박 입출항 정보는 실제 사용 데이터가 아니라 향후 확장 데이터입니다.
* 배포 후 Streamlit Cloud 링크 접속 테스트를 반드시 진행합니다.
* 발표 당일을 대비해 주요 화면 캡처와 데모 영상을 별도로 준비합니다.
